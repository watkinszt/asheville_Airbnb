# Buncombe County Airbnbs (2020)

Interactive Leaflet map visualizing Airbnb listings and zoning districts in Buncombe County, NC.

## Features
- Airbnb listings shown as Font Awesome icons  
  - Entire house  
  - Private room in house  
  - Other  
- Zoning districts styled with a 5-class choropleth (Purples)
- Popups showing property type
- Legend and scale bar
- CartoDB Light basemap

## Data Sources
- Inside Airbnb  
- City of Asheville Open Data  
- CartoDB
